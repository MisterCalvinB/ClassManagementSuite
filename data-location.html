    Bundled files differ from your data folder

    The app ships with default data files (the bundled version). One or more of those files differ from what is already in your data folder. For each file below, choose what to do:

    Keep — Do nothing. Your existing file stays exactly as it is. Choose this if you have customised the file and do not want to lose your changes.
    Replace — Overwrite your file with the bundled (default) version. Any changes you made to that file will be permanently lost. Choose this if you want the latest default content and do not need your current version.
    Rename & replace — Save your current file under a new name ending in _bak (e.g. wordDb_bak.js), then copy the bundled version in its place. This is the safest option: you get the updated default file while keeping a backup of your previous one.
